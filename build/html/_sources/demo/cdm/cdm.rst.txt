Convolution Directivity Model (CDM)
=====================================================

Demos related to the CDM.

Demo: `PalLineSrc_CDM_Demo1.m`
-----------------------------------------------

Compares the directivity of the audio sound generated by a PAL using the CDM.

Examples
^^^^^^^^^^^^^^^^^^^^^
Parameters and the codes for each example are given as follows.

Example 1
"""""""""""""""""""

============================== ==============================
Variable                            Value
------------------------------ ------------------------------
Aperture size (half-width)                   10 cm
Source profile                  Uniform (piston source)
Source shape                     Line
Audio frequency                 4 kHz
Ultrasound frequency            40 kHz
============================== ==============================

..  code-block:: matlab

    prf = SrcProfile('name', 'uniform');
    src = LineSrc('radius', 0.1, 'prf', prf);
    pal = PalSrc('audio_freq', 4e3, 'ultra_freq', 40e3, 'src', src);

Example 2
"""""""""""""""""""

============================== ==============================
Variable                            Value
------------------------------ ------------------------------
Aperture size (half-width)      5 cm
Source profile                  Steerable with an angle of :math:`15^\circ`
Source shape                     Line
Audio frequency                 1 kHz
Ultrasound frequency            40 kHz
============================== ==============================

..  code-block:: matlab

    prf = SrcProfile('name', 'steerable', 'phi', 15/180*pi);
    src = LineSrc('radius', 0.05, 'prf', prf);
    pal = PalSrc('audio_freq', 1e3, 'ultra_freq', 40e3, 'src', src);



Demo: `PalPlanarSrc_CDM_Demo1.m`
-----------------------------------------------
Calculate the directivity of the audio sound generated by a planar PAL in a 3D radiation problem using the CDM.

Examples
^^^^^^^^^^^^^^^^^^^^^
Parameters and the codes for each example are given as follows.

Example 1
"""""""""""""""""""

============================== ==============================
Variable                            Value
------------------------------ ------------------------------
Aperture size (half-width)        10 cm
Source profile                  Uniform (piston source)
Source shape                     Circle
Audio frequency                 4 kHz
Ultrasound frequency            40 kHz
============================== ==============================

..  code-block:: matlab

    prf = SrcProfile('name', 'uniform');
    src = CircSrc('radius', .1, 'prf', prf);
    pal = PalSrc('audio_freq', 4e3, 'ultra_freq', 40e3, 'src', src);

Example 2
"""""""""""""""""""

======================================          ============================================================
Variable                                            Value
--------------------------------------          ------------------------------------------------------------
Half width in the :math:`x`-direction               :math:`a_x = 0.1` m
Half width in the :math:`y`-direction               :math:`a_y = 0.05` m
Source profile                                  Steerable with an angle pair of :math:`(15^\circ, 0^\circ)`
Source shape                                    Rectangle
Audio frequency                                 1 kHz
Ultrasound frequency                            40 kHz
======================================          ============================================================

..  code-block:: matlab

    prf = SrcProfile('name', 'steerable', 'phi', 0, 'theta', 15/180*pi);
    src = RectSrc('prf', prf, 'ax', 0.1, 'ay', 0.05);
    pal = PalSrc('audio_freq', 1e3, 'ultra_freq', 40e3, 'src', src);
