3.2.1.2. BesselJLog_Rothwell.m¶

Calculate the log of the spherical Bessel function \(J_n(z)\) using the method proposed in Rothwell [Rot08].

3.2.1.2.1. Syntax¶

[J, J_prime] = BesselJLog_Rothwell(N, z)

3.2.1.2.2. Output arguments¶

  • J — \(\ln J_n(z)\)

  • J_prime — \(\ln J_n'(z)\)

3.2.1.2.3. Input arguments¶

  • N — maximal order

  • z — the argument \(z\)

3.2.1.2.4. Dependencies¶

  • ContFracLentz.m
