HankelH_Asym.m¶

Calculate the Hankel function \(H_m(z)\) using the asymptotic method.

See Eq. (10.17.5) in Olver et al. [ODL+].

Syntax¶

H = HankelH_Asym(m, z)

Output arguments¶

  • H — \(\ln H_m(z)\)

Input arguments¶

  • m — order

  • z — the argument \(z\)

(3)

(1)¶\[e^{i\pi} + 1 = 0\]
(2)¶\[p=f(x)\]
(3)¶\[H_m^{(1)}(z) \sim \sqrt{\frac{2}{\mathrm{i}\pi z}} \frac{\mathrm{e}^{\mathrm{i}z}}{\mathrm{i}^m}\]

Dependencies¶

NA
